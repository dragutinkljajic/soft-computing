# soft-computing
Soft computing project, Faculty of Technical Sciences 2016.

Generički projekat radjen za level 2 (ocenu 7 i 8).
Implementirano prepoznavanje regiona (objekata) koji se kreću kroz frejmove.
Implementirano prepoznavanje cifara putem neuronske mreže.
Implementirana detekcija linije putem regiona (metoda regionprops, donji levi i gornji desni ugao) kao i detekcija
linije Hough-ovom transformacijom.

Konačni test rezultati nad test podacima za level 2 su oko 73%. Poteškoće prepoznavanja različitih regiona zbog mnogobrojnih preklapanja.
Konačni test rezultati nad test podacima za level 1 su oko 97%.

Ukoliko konačna implementacija ne zadovoljava kriterijume za polaganje generičkog projekta level 2, projekat se može pokrenuti
nad test podacima za generički projekat level 1 koji zadovoljava sve kriterijume.
